# YT-DLP-GUI
Yet another GUI for  YT-DLP   

This version is just a simple Python + Tkinter GUI to fetch all available formats, choose one, then it downloads it. 

Needs miniconda installed, an environment ( ytdlp ) setup and install PIP into it.  Note that its its supposed to do the setup on its own if you have conda installed but its not working yet. However if you setup the env manually and add pip first, it works just fine. It also uses the new(ish) JS back end requirements that YTDLP now has, and fetches it from the GitHub pages on each run so its not stale.
