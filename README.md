# YT-DLP-GUI
Yet another GUI for  YT-DLP   

This version is just a simple Python + Tkinter GUI to fetch all available formats, choose one, then it downloads it. 

Needs miniconda installed, an environment ( ytdlp ) setup and install PIP into it.  ( its supposed to do the setup on its own, not working yet, but if you setup the env manually and add pip first, it works fine )
Uses the new(ish) JS back end requirements that YTDLP now has, and fetches it from the GitHub pages on each run so its not stale.
